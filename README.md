# Tech stack

- Svelte 3
- TypeScript

# Deploying Tutorial

1. Clone this repo
2. Adjust `email`, `domain`, and `data_path` variables in scripts/init-letsencrypt.sh
3. Run `docker-compose up --build -d --remove-orphans`
